# SpartaCSS

SpartaCSS is RedSpartan Systems' independent, reusable design system — design
tokens, a base reset/layout layer, a core set of UI components, an icon
system, and a notifications feature module.

**Status:** early / pre-extraction. The package is not yet installable —
source extraction and packaging from the existing v1 implementation have not
started. See `docs/adr/0001-package-architecture.md` for the architecture
this repository is being built against.
